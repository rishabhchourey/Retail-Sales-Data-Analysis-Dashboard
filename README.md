# 📊 Sales Performance Dashboard

### 🚀 End-to-End Sales Data Analysis using SQL, Excel & Power BI

---

## 📌 Project Overview

This project focuses on performing end-to-end sales data analysis using SQL, Excel, and Power BI. The goal was to clean messy data and convert it into meaningful insights for business decision-making.

---

## 🧹 Data Cleaning Process

The dataset contained missing values, duplicate records, and inconsistent formats. The following steps were performed:

* Removed duplicate records using SQL
* Handled NULL values appropriately
* Standardized data formats (dates, text, numbers)
* Cleaned and structured data using Excel

---

## 🛠️ Tools & Technologies Used

* SQL (MySQL)
* Microsoft Excel
* Power BI

---

## 📊 Dashboard Features

The Power BI dashboard provides the following insights:

* 📈 Monthly Sales Trend Analysis
* 🧾 Payment Status Analysis (Success / Failed)
* 👥 Top Customers based on revenue
* 🌍 City-wise filtering
* 📅 Date range filtering

---

## 📈 Key Insights

* Identified peak sales months
* Found top contributing customers
* Analyzed payment success vs failure rate
* Compared sales performance across different cities

---

## 🎯 Conclusion

This project demonstrates the complete data analysis workflow — from raw data cleaning to building an interactive dashboard. It highlights the importance of data preprocessing and visualization in making data-driven business decisions.
